# Week 2 — Competency 2: Code literacy and documentation

Observations and reflections for HCDE 530.

---

## Code literacy

This week we **connected the project to GitHub**, which made the coursework feel “real” in the sense that code and files live somewhere shared and versioned—not only on my machine.

We **studied `demo_word_count.py`**, the script our professor prepared. Walking through the codes is a concrete way to build literacy: you see how a small program is structured (load data, define a helper, loop, summarize) without having to invent all of it from scratch.

I **added comments that label sections of the code** (what each block is for). That exercise forced me to read line-by-line and decide what “chunk” of logic belongs together—similar to annotating a journey map or flow, but for text the computer runs.

The hardest part for me was **deciding where one section ended and the next began**, not writing a short label once I had a chunk. I don’t have much coding background yet, so “natural” breakpoints in code are less obvious than in a research write-up or a deck outline.

I also **used Cursor to generate synthetic app reviews and a short Python script** that summarizes them (word counts and stats). That sits at the edge of “literacy”: I’m still responsible for judging whether the output matches the assignment and whether the script is readable and correct, even when the first draft comes from a tool.

---

## Documentation

**Comments as documentation:** I noticed that **comments show up in a way that “pops out”** in the editor (e.g. in the outline or structure view—depending on Cursor/IDE behavior). That made documentation feel less like optional decoration and more like something the environment surfaces—similar to how headings help readers scan a doc.

**AI-assisted drafts:** Asking Cursor to draft synthetic data and script code raises a documentation question too: the **script and any README or inline comments** are what explain to *me* (and graders) what the artifact is and how to run it—whether I wrote every line by hand or not.
